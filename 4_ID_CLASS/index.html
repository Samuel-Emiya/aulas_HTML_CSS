ID e CLASS

ID

O ID funciona como CPF. Assim como só posso ter um CPF, cada elemento só pode ter um valor ID. o CPF é único, ou seja, ninguem pode ter um CPF igual ao meu. Isso significa que o ID também é único, então não é possível repetir o mesmo valor duas vezes

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode ultilizar o mesmo valor para vários elementos
